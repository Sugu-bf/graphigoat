Notes:
- The Magic Lite client page is marked noindex,nofollow and is not linked from the homepage.
- It is still a static page. Anyone with the exact URL can open it unless your host adds password protection.
- Workstream comments, selections, and budget entries are saved in each visitor's browser storage. They are not shared between visitors without a backend.
